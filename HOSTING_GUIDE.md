# Руководство по хостингу и размещению на Beget
> Инструкция для заказчика: какой тариф выбрать, как загрузить сайт, как поддерживать

---

## 1. Выбор тарифа на Beget

Для статического сайта (HTML + CSS + JS, без базы данных) подходит **любой минимальный тариф**.

### Рекомендуемый тариф

| Параметр | Требование |
|----------|------------|
| Тип хостинга | Виртуальный хостинг (shared) |
| Минимальный тариф | "Start" или "Premium" (если планируете другие проекты) |
| Дисковое пространство | 1-3 ГБ более чем достаточно для сайта пиломатериалов |
| Трафик | Любой безлимитный |
| PHP / MySQL | Не нужны для этого сайта |
| SSL (HTTPS) | Обязателен. На Beget выдаётся бесплатно через Let's Encrypt |

### Примерные цены (ориентировочно, уточняйте на beget.ru)
- Start: ~150-200 ₽/мес
- Premium: ~300-400 ₽/мес

**Для сайта пиломатериалов достаточно тарифа Start.**

## 2. Регистрация и первичная настройка

### Шаг 1. Регистрация аккаунта
1. Перейти на https://beget.ru
2. Нажать "Зарегистрироваться"
3. Указать email, телефон, придумать пароль
4. Подтвердить email

### Шаг 2. Добавление домена

**Вариант А: У вас уже есть домен (например, pilorama-ivanovo.ru)**
1. В панели управления Beget перейти в раздел "Домены" → "DNS"
2. Узнать DNS-сервера Beget (обычно: ns1.beget.com, ns2.beget.com, ns3.beget.com, ns4.beget.com)
3. Зайти в личный кабинет регистратора домена (где покупали домен)
4. Сменить DNS-серверы на Beget-овские
5. Подождать 2-24 часа (распространение DNS)

**Вариант Б: Нужно купить домен**
1. В панели Beget: "Домены" → "Зарегистрировать новый домен"
2. Ввести желаемое название, выбрать зону (.ru, .рф, .com)
3. Оплатить регистрацию (~200-800 ₽/год в зависимости от зоны)

### Шаг 3. Включение SSL (HTTPS)
1. Панель управления → "Сайты" → выбрать свой сайт
2. Нажать "SSL" → "Установить Let's Encrypt"
3. Выбрать домен, подтвердить
4. Сертификат активируется в течение 5-15 минут
5. **Важно:** Весь сайт должен открываться по HTTPS (без ошибок смешанного контента)

## 3. Загрузка файлов сайта

### Способ 1. Файловый менеджер Beget (проще)

1. Войти в панель управления Beget
2. Перейти в раздел "Файловый менеджер"
3. Открыть папку `/public_html/` (или `/www/имя_домена/` если несколько сайтов)
4. Удалить стандартный `index.html` (заглушка Beget)
5. Загрузить архив (ZIP) со всеми файлами сайта
6. Распаковать архив внутри менеджера
7. Итоговая структура должна быть:
   ```
   /public_html/
     ├── index.html
     ├── css/
     │   └── style.css
     ├── js/
     │   └── script.js
     ├── images/
     │   ├── hero.jpg
     │   ├── products/
     │   └── base/
     └── favicon.ico
   ```

### Способ 2. FTP-клиент (удобнее для обновлений)

| Параметр | Где взять |
|----------|-----------|
| FTP-сервер | Указан в панели Beget (обычно: имя_аккаунта.beget.tech) |
| FTP-пользователь | Тот же логин, что и от панели |
| FTP-пароль | Тот же пароль, что и от панели |
| Порт | 21 (стандартный) |

**Программы для FTP:**
- **FileZilla** (бесплатная, Windows/Mac/Linux)
- **WinSCP** (Windows)

**Инструкция:**
1. Скачать и установить FileZilla
2. Файл → Менеджер сайтов → Новый сайт
3. Ввести данные FTP из панели Beget
4. Подключиться
5. В левой панели — файлы на вашем компьютере, в правой — сервер
6. Перетащить папку сайта в `/public_html/`

## 4. Структура файлов сайта (как будет выглядеть)

```
/public_html/
├── index.html              # Главная и единственная страница (лендинг)
├── assets/
│   ├── css/
│   │   ├── variables.css   # Дизайн-токены (цвета, шрифты)
│   │   ├── theme.css       # Тема и компоненты
│   │   └── main.css        # Основные стили секций
│   ├── js/
│   │   ├── main.js         # Мобильное меню, якоря, lazy loading
│   │   └── map.js          # Инициализация Яндекс.Карты
│   └── fonts/
│       ├── playfair-display-*.woff2
│       └── inter-*.woff2
├── images/
│   ├── hero.jpg            # Главное фото (1920×1080)
│   ├── base/
│   │   ├── base-1.jpg      # Фото базы
│   │   ├── base-2.jpg
│   │   └── ...
│   └── products/
│       ├── brus-100x150.jpg
│       ├── doska-obreznaya.jpg
│       └── ...
└── favicon.ico             # Иконка вкладки
```

**Важно:** Все пути в коде относительные (`./images/hero.jpg`), абсолютные не нужны.

## 5. Подключение Яндекс.Карт

1. Получить бесплатный API-ключ: https://developer.tech.yandex.ru/services/
2. Выбрать "JavaScript API и HTTP Геокодер"
3. Ввести адрес сайта (например, `https://pilorama-ivanovo.ru`)
4. Скопировать ключ вида `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`
5. Передать ключ разработчику — он встроит в `map.js`

**Ограничения бесплатного ключа:**
- До 25 000 запросов в сутки (для сайта пиломатериалов — хватит с запасом)
- Обязательно указание адреса сайта при регистрации ключа

## 6. Поддержка и обновление сайта

### Часто обновляется (раз в 1-3 месяца)

| Что обновлять | Как | Сложность |
|---------------|-----|-----------|
| Цены на пиломатериалы | Редактировать `index.html` — меняются цифры в карточках | Низкая |
| Телефон / контакты | Редактировать `index.html` | Низкая |
| Фото новой продукции | Загрузить в `images/products/`, добавить блок в `index.html` | Средняя |

### Редко обновляется (раз в год или реже)

| Что обновлять | Как | Сложность |
|---------------|-----|-----------|
| Реквизиты ИП | Редактировать `index.html` | Низкая |
| Фото базы | Заменить файлы в `images/base/` | Низкая |
| Тексты / преимущества | Редактировать `index.html` | Низкая |

### Как редактировать файлы без программиста

**Для простых правок (цены, телефон):**
1. Скачать `index.html` с сервера через FTP
2. Открыть в обычном текстовом редакторе (Notepad++, VS Code)
3. Найти нужный текст (Ctrl+F)
4. Изменить, сохранить
5. Загрузить обратно на сервер через FTP

**Для загрузки новых фото:**
1. Сжать фото через https://tinypng.com (обязательно!)
2. Загрузить в нужную папку через FTP или файловый менеджер
3. Если добавляете новую позицию — потребуется правка `index.html`

## 7. Резервное копирование

### Автоматическое (настроить в Beget)
1. Панель управления → "Резервные копии"
2. Включить автоматическое резервное копирование (1 раз в неделю)
3. Хранить последние 3-4 копии

### Ручное (перед каждым обновлением)
1. В FTP-клиенте: выделить все файлы сайта
2. Скачать на компьютер в папку с датой (например, `backup-2026-05-10`)
3. Только после этого вносить изменения на сервер

## 8. Проверка работоспособности после размещения

| Чек-пункт | Как проверить |
|-----------|---------------|
| Сайт открывается | Ввести домен в браузере, должна загрузиться главная страница |
| HTTPS работает | В адресной строке должен быть замок (без предупреждений) |
| Карта загружается | Прокрутить до секции контактов — карта не пустая, есть маркер |
| Фото загружаются | Нет битых картинок (крестиков), нет фото, открывающихся в новой вкладке |
| Мобильная версия | Открыть сайт на телефоне или в Chrome: F12 → Toggle Device Toolbar |
| Скорость загрузки | https://pagespeed.web.dev/ — оценка должна быть зелёной (90+) |
| Кнопка "Позвонить" | На мобильном нажать — должен сразу предложить звонок |

## 10. Кому обращаться в случае проблем

| Проблема | Куда обращаться |
|----------|-----------------|
| Сайт не открывается, ошибка 500/502 | В поддержку Beget (чат/тикет в панели) |
| Забыл пароль от панели | Восстановление пароля на beget.ru |
| Нужно изменить дизайн / добавить секцию | К разработчику |
| Карта перестала работать | Проверить API-ключ в кабинете Яндекса, затем к разработчику |
| SSL пропал / браузер ругается | Переустановить сертификат в панели Beget |
