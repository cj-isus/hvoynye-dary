## 4. Продукция и цены

> Сайт не тянет данные из БД — всё встраивается статически. Цены обновляются вручную.

### Для каждой позиции нужно:

| Поле | Пояснение | Пример |
|------|-----------|--------|
| Название | Кратко, понятно покупателю | Брус 100×150×6000 мм |
| Категория | Для группировки | Брус, Доска, Вагонка, Блок-хаус |
| Цена за м³ | Основная цена | от 12 500 ₽/м³ |
| Цена за штуку | Если удобнее считать так | от 750 ₽/шт |
| Порода дерева | Сосна, Ель, Лиственница, Осина и т.д. | Сосна |
| Влажность | Естественная / камерная сушка | Естественная |
| Сорт | 1-2, 1-3, Прима, АВ | 1-2 |
| Краткое описание | 1 строка особенностей | Строганный, сухостой |
| Фото | Смотри раздел "Фото" ниже | 1-3 фото на товар |

### Рекомендуемый минимум позиций для запуска:
- 4-6 позиций = достаточно для старта
- 8-12 позиций = оптимально
- 15+ позиций = если ассортимент широкий

## 5. Фото и изображения

### 5.1. Фото базы / производства (секция "О нас / Производство")
> Нужно для доверия. Доказывают, что вы реальные, а не перекуп.

| Сюжет | Количество | Требования |
|-------|------------|------------|
| Общий план базы / склада | 2-3 шт. | Горизонтальные, дневной свет, чистая композиция |
| Пилорама / станки в работе | 2-3 шт. | Действие, стружка/пыль — атмосфера производства |
| Сортировка / укладка пиломатериалов | 2-3 шт. | Показывает качество контроля |
| Погрузка / техника | 1-2 шт. | Если есть вилочный погрузчик — показывает масштаб |
| Сырьё (брёвна) | 1-2 шт. | Доказывает прямое производство |

### 5.2. Фото продукции (карточки товаров)

| Требование | Параметр |
|------------|----------|
| Формат | JPG, качество 80-90% |
| Соотношение сторон | 4:3 или 1:1 (квадрат) — единообразно |
| Минимальный размер | 800×600 px |
| Оптимальный размер | 1200×900 px |
| Фон | Лучше нейтральный (не ковёр/трава). Паллеты, склад, ровная укладка |
| Освещение | Дневное, без резких теней |
| Количество | Минимум 1 фото на товар. Оптимально — 2 (общий план + деталь текстуры) |

### 5.3. Hero-фото (главный экран)

| Требование | Параметр |
|------------|----------|
| Формат | Горизонтальное, широкое (16:9 или 21:9) |
| Разрешение | Минимум 1920×1080 px |
| Сюжет | Либо атмосферное фото базы на закате/рассвете, либо крупный план качественного пиломатериала |
| Обработка | Можно слегка затемнить (мы наложим градиент #1f1f29 поверх) |

## 8. Порядок передачи

1. **Сначала** — тексты (п.1, п.2, п.3, п.7). На их основе собираем структуру.
2. **Параллельно** — фото базы (п.5.1). Можно с телефона, главное — чёткость и свет.
3. **Затем** — таблица цен и продукции (п.4). Удобно в Excel / Google Таблицах.
4. **В конце** — фото продукции (п.5.2). Лучше делать в один день при хорошем свете.

## 9. Что произойдёт, если что-то не предоставить

| Если не дать | Результат |
|--------------|-----------|
| Фото базы | Секция "Производство" будет скрыта или заполнена стоковыми фото (не рекомендуется) |
| Реквизиты | Подвал будет без блока "Реквизиты ИП" — меньше доверия у B2B-клиентов |
| Цены | Сайт запустится с текстом "Цены по запросу" — допустимо, но хуже конверсия |
| УТП / преимущества | Hero-блок будет шаблонным — упущенная возможность цеплять клиента |
