# План контента — одностраничный лендинг
> Структура каждой секции: что в ней, какой контент, примеры формулировок, CTA

---

## Общая структура страницы (порядок сверху вниз)

1. **Навигация (sticky)**
2. **Hero** — главный экран
3. **Преимущества** — почему мы
4. **Продукция** — каталог с ценами
5. **Фото базы / производства** — галерея
6. **Контакты + Карта**
7. **Футер** — реквизиты ИП

**На мобильном:** всё то же самое, Sticky CTA добавляется поверх контента внизу экрана.

---

## 1. Навигация (Sticky Header)

### Структура
- **Лево:** Логотип (текстовый в Playfair Display) или SVG-логотип
- **Центр (desktop):** Якорные ссылки — Продукция, База, Контакты
- **Право (desktop):** Телефон (кликабельный, Inter 15px, weight 500) + кнопка "Позвонить" (Solid Dark Button)
- **Мобильно:** Бургер-меню, при открытии — панель с якорями + телефон крупно

### Поведение
- При скролле вниз — навигация превращается в Blurred Nav Item: фон `rgba(249,250,247,0.9)`, `backdrop-filter: blur(12px)`, radius 50px, тень `--shadow-sm`
- На мобильном — плавающая панель снизу или фиксированный top-bar

## 2. Hero Section

### Цель секции
За 3 секунды понятно: кто, где, что продаёт. Оставить телефон или прокрутить вниз.

### Фон
- Фото базы / крупный план пиломатериала (1920×1080)
- Градиент затемнения справа налево: `linear-gradient(90deg, rgba(31,31,41,0.92) 0%, rgba(31,31,41,0.5) 50%, rgba(31,31,41,0.2) 100%)`
- Текст расположен слева (читается на тёмном градиенте)

### Контент (слева направо)

| Элемент | Текст / Данные | Стиль |
|---------|---------------|-------|
| Микро-лейбл | "Производство пиломатериалов · [Город]" | Inter, 13px, `rgba(255,255,255,0.6)`, uppercase, tracking 0.05em |
| Заголовок H1 | "Пиломатериалы от производителя" | Playfair Display, fluid-display, `#ffffff` |
| Подзаголовок | "Брус, доска, вагонка и блок-хаус. Сосна и ель. Доставка по области." | Inter, subheading, `rgba(255,255,255,0.8)` |
| Hero Overlay Card (под текстом) | Телефон крупно + "Работаем: [дни] [часы]" + Адрес кратко | Фон `rgba(222,226,222,0.16)`, radius 24px, blur backdrop |
| CTA-кнопки | "Позвонить" (Solid Dark Button) + "Написать в WhatsApp" (Outlined, border `#41a1cf`) | Inter, 16px |

### Поведение на мобильном
- Заголовок уменьшается до `clamp(32px, 8vw, 54px)`
- Hero Overlay Card занимает 100% ширины под текстом
- Кнопки стопроцентной ширины, друг под другом
- Высота секции: `100vh` (полный экран)

## 3. Секция "Преимущества" (почему мы)

### Цель
Убедить, что вы реальный производитель, а не перекуп. 4-5 коротких пункта.

### Фон
- `#ffffff` (canvas-white)

### Контент

| № | Иконка | Заголовок | Подпись |
|---|--------|-----------|---------|
| 1 | 🌲 (outline) | Собственная пилорама | Производим сырьё с 20XX года. Контроль качества на каждом этапе. |
| 2 | 🚚 (outline) | Доставка по области | Собственный транспорт. Разгрузка манипулятором. |
| 3 | 📏 (outline) | Честный сорт | Замеряем перед отгрузкой. Соответствие ГОСТ. |
| 4 | 💰 (outline) | Цена от производителя | Без посредников. Опт и розница. Объёмные скидки. |

### Стиль
- Сетка: 4 колонки (desktop), 2 колонки (tablet), 1 колонка (mobile)
- Карточка: без фона, иконка 32px в `--color-cofounder-blue`, заголовок Inter 18px weight 500, подпись Inter 15px `#646464`
- Иконки — outline, stroke 1.5px, монохромные
- Gap между карточками: 32px
- Верхний/нижний padding секции: 80px

## 4. Секция "Продукция"

### Цель
Показать ассортимент и цену. Каждый товар — визуальная карточка, не таблица.

### Фон
- `#fefffc` (off-white) — мягкое отделение от предыдущей секции

### Заголовок секции
- "Продукция и цены" — Playfair Display, heading-lg, `#171717`
- Под ним линия-акцент 40px шириной, 2px высотой, цвет `#C4A882` (wood-warm)
- Подзаголовок: "Цены актуальны на [месяц год]. Уточняйте по телефону." — Inter, 15px, `#646464`

### Сетка товаров
- CSS Grid, `auto-fill`, `minmax(280px, 1fr)`, gap 24px
- Desktop: 3-4 карточки в ряд
- Tablet: 2 карточки
- Mobile: 1 карточка

### Структура карточки товара

```
┌─────────────────────────────┐
│  [Фото товара 4:3]          │
├─────────────────────────────┤
│  Название                   │  Inter 18px, #171717
│  Порода · Сорт · Влажность  │  Inter 13px, #646464
│                             │
│  от 12 500 ₽/м³             │  Inter 24px, #0081c0, weight 600
│                             │
│  [Узнать цену]              │  Outlined Action Button
└─────────────────────────────┘
```

### Данные для каждой карточки (пример)

| Поле | Пример |
|------|--------|
| Название | Брус 100×150×6000 мм |
| Порода | Сосна |
| Сорт | 1-2 |
| Влажность | Естественная |
| Цена за м³ | от 12 500 ₽ |
| Цена за штуку | от 750 ₽ |
| Кнопка | "Узнать точную цену" → звонок или WhatsApp |

### Hover-эффект карточки
- Подъём `translateY(-4px)`
- Тень усиливается до `--shadow-subtle-3`
- Верхняя граница 2px `#C4A882` появляется плавно
- Фото слегка увеличивается `scale(1.03)`

### Поведение на мобильном
- Карточки вертикальные, 100% ширины
- Цена крупнее, кнопка "Узнать цену" на всю ширину карточки
- Отступы увеличены для удобства пальцевого скролла

## 5. Секция "Фото базы"

### Цель
Доказательство. Живые фото склада и производства создают доверие лучше любых текстов.

### Фон
- `#ffffff` (canvas-white)

### Заголовок секции
- "Производство в [Городе]" — Playfair Display, heading-lg
- Подзаголовок: "Собственная пилорама, склад сырья и готовой продукции" — Inter, 18px, `#646464`

### Галерея
- CSS Grid, 4 колонки (desktop), 3 (tablet), 2 (mobile)
- Aspect-ratio: 4/3 или 3/4 (чередование для визуального ритма)
- Radius: 12px
- Gap: 16px
- Фото открываются в лайтбоксе (без перехода на новую страницу)

### Подписи к фото
- При hover появляется градиент снизу + текст
- Текст: Inter 13px, `#ffffff` — краткое описание ("Сортировка бруса", "Погрузка", "Склад сырья")

### Минимальный набор фото (7 штук)
1. Общий план базы / склада (широкий, 16:9 — может занять 2 колонки в сетке)
2. Пилорама в работе
3. Сортировка / укладка
4. Готовый брус / доска крупным планом
5. Склад брёвен / сырьё
6. Техника / погрузка
7. Готовая продукция на паллетах

## 6. Секция "Контакты + Карта"

### Цель
Всё для связи и приезда. Телефон, адрес, часы — крупно. Карта — рядом.

### Фон
- `#fefffc` (off-white)
- Верхний border 1px `#dee2de` (тонкое разделение)

### Layout (desktop)
- 2 колонки: 40% контакты / 60% карта

### Левая колонка — Контакты (Elevated Content Card)

| Блок | Содержимое | Стиль |
|------|-----------|-------|
| Заголовок | "Контакты" | Playfair Display, heading, `#171717` |
| Телефон | +7 (XXX) XXX-XX-XX | Inter, 28px, weight 500, `#0081c0` — кликабельный |
| Адрес | Полный адрес с ориентиром | Inter, 18px, `#2c2c2c` |
| Режим работы | [Дни], [Часы] | Inter, 16px, `#646464` |
| Кнопки | "Позвонить" + "WhatsApp" | Solid Dark + Outlined |
| Примечание | "Звоните или пишите — ответим за 10 минут" | Inter, 13px, `#646464` |

### Правая колонка — Яндекс.Карта
- Размер: 100% ширины колонки, высота 450px
- Тема: тёмная (соответствует Night Sky)
- Маркер: цвет `#0081c0`
- Балун: название ИП + телефон

### Мобильная версия
- Карта сверху: 100% ширины, 300px высота
- Контакты под картой: телефон ещё крупнее (32px), кнопки на всю ширину
- При нажатии на телефон — сразу вызов

## 7. Футер

### Цель
Реквизиты для B2B-клиентов и юридическая прозрачность.

### Фон
- `#1f1f29` (night-sky)
- Текст: `#ffffff` и `rgba(255,255,255,0.6)`

### Контент

| Блок | Содержимое | Стиль |
|------|-----------|-------|
| Название ИП | Иванов Иван Иванович ИП | Playfair Display, 18px, `#ffffff` |
| Реквизиты | ИНН: XXXXXXXXXX / ОГРНИП: XXXXXXXXXXXXXXX | Inter, 13px, `rgba(255,255,255,0.6)` |
| Доп. реквизиты | Банк, БИК, р/с, к/с (если нужны) | Inter, 13px, `rgba(255,255,255,0.6)` |
| Копирайт | © 20XX ИП Иванов И.И. | Inter, 13px, `rgba(255,255,255,0.4)` |

### На мобильном
- Всё в одну колонку, по центру
- Отступы увеличены для читаемости

## 8. Sticky Mobile CTA (только на мобильных < 768px)

### Позиция
- Fixed, bottom: 0
- Фон: `#ffffff`
- Тень: `--shadow-sm`
- Border-top: 1px `#dee2de`

### Содержимое
- 2 кнопки в ряд:
  - **"Позвонить"** — Solid Dark Button, 50% ширины, с иконкой трубки
  - **"WhatsApp"** — Outlined Action Button (border `#41a1cf`), 50% ширины, с иконкой
- Safe area: `padding-bottom: env(safe-area-inset-bottom)` (для iPhone с чёлкой)
